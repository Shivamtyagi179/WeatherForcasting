# 🌦️ Weather Forecasting Website

A modern and responsive weather forecasting web application that provides real-time weather updates and future forecasts for any location worldwide.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💧 Humidity and wind speed details
* ☁️ Weather conditions (Cloudy, Rainy, Sunny, etc.)
* 📅 Multi-day weather forecast
* 📱 Fully responsive design (Mobile + Desktop)
* ⚡ Fast and lightweight performance

---

## 🛠️ Tech Stack

**Frontend:**

* HTML5
* CSS3
* JavaScript

**API:**

* Weather API (e.g., OpenWeatherMap or similar)

**Tools & Platform:**

* Git & GitHub

1. Get your API key from your weather provider (e.g., OpenWeatherMap)
2. Replace the API key in `script.js`

